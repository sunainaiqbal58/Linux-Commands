# Basic Linux Commands

This document contains some of the basic Linux commands that I have practiced while learning Linux for Networking and Cyber Security.

---

## 1. pwd (Print Working Directory)

 Purpose:
Displays the current working directory.
Syntax:
```bash
pwd
```

Example:
```bash
pwd
```

**Output:**
```text
/home/sunaina/Documents
```

---

 2. ls (List Directory Contents)

Purpose:
Lists the files and folders available in the current directory.

Syntax:
```bash
ls
```
Example:
```bash
ls
```

 Output:
```text
Documents  Downloads  Desktop  Pictures
```

---

 3. cd (Change Directory)

Purpose:
Changes the current working directory.

Syntax:
```bash
cd <directory_name>
```
Example:
```bash
cd Documents
```
---
 4. mkdir (Make Directory)

Purpose:
Creates a new directory.
Syntax:
```bash
mkdir <directory_name>
```

Example:
```bash
mkdir CyberSecurity
```
---

 5. touch

Purpose:
Creates a new empty file.

Syntax:
```bash
touch <file_name>
```
Example:
```bash
touch notes.txt
```
---

6. rm (Remove)

Purpose:
Deletes a file.

Syntax:
```bash
rm <file_name>
```
Example:
```bash
rm notes.txt
```
> **Note:** Be careful while using the `rm` command because deleted files cannot be recovered easily.

---

 7. cp (Copy)

Purpose:
Copies a file from one location to another.

Syntax:
```bash
cp <source> <destination>
```

Example:
```bash
cp file1.txt file2.txt
```
---

 8. mv (Move)

Purpose:
Moves or renames a file.

Syntax:
```bash
mv <source> <destination>
```

Example:
```bash
mv old.txt new.txt
```
